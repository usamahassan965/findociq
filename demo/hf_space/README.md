# FinDocIQ — live demo

Ask questions over financial documents — including the charts and tables that
text-only RAG can't see. Answers are generated by a vision-language model
reading the actual retrieved page images, with per-claim citations.

**Loaded documents:** Berkshire Hathaway 2024 shareholder letter · EIA
Short-Term Energy Outlook.

**Demo scope:** this free CPU Space runs the text-retrieval lane
(BGE-small dense retrieval) of the full pipeline. The complete system adds
ColQwen2.5 late-interaction visual retrieval fused via Reciprocal Rank
Fusion over Qdrant multivector collections.

Full source, architecture, and evaluation:
**https://github.com/usamahassan965/findociq**
